## 快速开始
前置：Node.js 16+、npm（或 yarn）

```bash
cd trip-manager/backend
npm install
npm run init-db
npm run start
```

```bash
cd trip-manager/frontend
npm install
npm run dev
```

访问：
- http://localhost:5173
- 默认账号：`admin` / `admin123`

# 添加新研学团组

基于提供的参数添加新团组：$ARGUMENTS

请执行以下操作：
1. 解析参数：团组名称、类型、开始日期、结束日期、学生人数、老师人数
2. 计算行程天数（结束日期-开始日期+1）
3. 为团组分配颜色
4. 通过API创建新团组
5. 确认团组创建成功
6. 验证基础活动卡片是否自动生成
7. 提供团组创建结果报告

示例用法：/add-group 测试学校 primary 2025-09-20 2025-09-24 35 3
